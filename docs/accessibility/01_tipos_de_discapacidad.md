# Tipos de discapacidad

| # | Tipo | Subtipos relevantes |
|---|---|---|
| 1 | **Visual total** | Ceguera — depende 100% de lector de pantalla |
| 2 | **Baja visión** | Usa zoom, contraste alto, fuente grande |
| 3 | **Daltonismo** | Deuteranopía (rojo/verde), Protanopía, Tritanopía (azul/amarillo) |
| 4 | **Fotosensibilidad** | Epilepsia por parpadeo, migrañas por movimiento |
| 5 | **Auditiva total** | Sordera — no percibe audio |
| 6 | **Hipoacusia** | Escucha con ayuda, necesita captions |
| 7 | **Motora severa** | Tetraplejia, usa switch, eye-tracking, teclado solo |
| 8 | **Motora parcial** | Temblor (Parkinson), un solo brazo, precisión reducida |
| 9 | **Cognitiva** | Dislexia, TDAH, discalculia |
| 10 | **Intelectual** | Comprensión lectora limitada, necesita lenguaje simple |
| 11 | **Neurológica** | Ansiedad por tiempo, procesamiento lento |
| 12 | **Del habla** | Tartamudez, mutismo (relevante en voice UI o chat) |
| 13 | **Temporal/Situacional** | Brazo roto, sol en pantalla, manos ocupadas |
