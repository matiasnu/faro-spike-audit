# Principio 1: Perceptible

> ¿Puedo percibir el contenido?

| Problema | Discapacidad afectada |
|---|---|
| Imagen sin `alt` | Visual total |
| `alt` genérico ("foto", "image1.jpg") | Visual total |
| Imagen decorativa con `alt` descriptivo (ruido) | Visual total |
| Video sin captions/subtítulos | Auditiva |
| Audio sin transcripción | Auditiva |
| Solo color para transmitir info (rojo=error) | Daltonismo |
| Contraste texto/fondo insuficiente (<4.5:1) | Baja visión, daltonismo |
| Contraste de componentes UI (<3:1) | Baja visión |
| Texto en imagen (no escalable, no seleccionable) | Baja visión, dislexia |
| Sin control de audio autoplay | Auditiva (interfiere con lector) |
| Contenido no reflow a 320px (scroll horizontal) | Baja visión (zoom) |
| Tamaño de fuente no escalable (px fijos, no rem) | Baja visión |
| Iconos sin label textual | Visual total, cognitiva |
