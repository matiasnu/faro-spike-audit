# Principio 3: Comprensible

> ¿Puedo entender?

| Problema | Discapacidad afectada |
|---|---|
| Sin `lang` en `<html>` | Visual total (lector de pantalla) |
| Cambio de idioma inline sin `lang` en el elemento | Visual total |
| Cambio inesperado de contexto al hacer focus | Cognitiva, motora |
| Cambio inesperado al seleccionar un `<select>` | Cognitiva |
| Inputs sin `<label>` (solo placeholder) | Visual total, cognitiva |
| Campos requeridos sin indicación textual | Cognitiva |
| Errores solo por color sin texto descriptivo | Daltonismo, cognitiva |
| Sin sugerencia de corrección en errores | Cognitiva, intelectual |
| Sin confirmación antes de acción destructiva | Cognitiva, motora |
| Lenguaje complejo / jerga sin glosario | Intelectual, cognitiva |
| Instrucciones ambiguas ("haz clic en el ícono") | Visual total |
| Sin resumen ni tabla de contenidos en páginas largas | TDAH, cognitiva |
