# Principio 2: Operable

> ¿Puedo interactuar?

| Problema | Discapacidad afectada |
|---|---|
| Sin navegación por teclado (`Tab`) | Motora severa |
| `outline: none` en `:focus` (foco invisible) | Motora severa, baja visión |
| Keyboard trap (modal sin escape) | Motora severa |
| Solo interacción por hover/mouse | Motora severa |
| Custom widget sin soporte de teclas (Enter, Space, flechas) | Motora severa |
| Target de click < 24x24px (WCAG 2.2) | Motora parcial, temblor |
| Timeout sin advertencia ni extensión | Cognitiva, motora |
| Contenido que parpadea >3 veces/seg | Fotosensibilidad |
| Sin "skip to main content" | Motora severa, visual total |
| Sin título único por página (`<title>`) | Visual total |
| Orden de foco no lógico (saltos raros) | Motora severa, visual total |
| Links sin propósito claro ("click aquí") | Visual total |
| Gestos complejos sin alternativa simple | Motora severa |
| Animación activada por movimiento del dispositivo | Motora severa |
